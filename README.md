# ⏱️ SCT_WT_2 – Stopwatch Web Application

## 📖 Overview

**SCT_WT_2** is a simple and interactive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**.
The application allows users to measure elapsed time with essential stopwatch controls and record lap times.

The project demonstrates core front-end development concepts such as **DOM manipulation, event handling, and timing functions in JavaScript**.

---

## ✨ Features

* ▶️ **Start Timer** – Begins the stopwatch
* ⏸️ **Pause Timer** – Temporarily stops the timer
* 🔄 **Reset Timer** – Resets the stopwatch to `00:00:00`
* 🏁 **Lap Recording** – Stores lap times during the session
* 📱 **Responsive Design** – Works across desktop, tablet, and mobile devices
* 🎨 **Clean UI** – Minimal and professional user interface

---

## 🛠️ Technologies Used

* **HTML5** – Provides the structure of the application
* **CSS3** – Handles layout, styling, and responsive design
* **JavaScript (ES6)** – Implements stopwatch logic and user interactions

---

## ⚙️ How It Works

1. Clicking **Start** begins the timer using JavaScript's `setInterval()` function.
2. The timer updates every second and displays the elapsed time.
3. **Pause** stops the timer without resetting the time.
4. **Reset** clears the timer and lap records.
5. **Lap** records the current stopwatch time and displays it in the lap list.
